Tugas Pagination

Tugas Pemorgraman Web Week-2 membuat fitur pagination dari bootstrap, disini saya membuat website album foto pemain barcelona dengan pagination, jadi terdapat 3 halaman dan setiap halamannya berisi foto pemain FC Barcelona, pagination dibuat interaktif dengan ketika tombol halaman berapapun dipencet maka halamannya otomatis akan berubah, dan ketika tombol "Previous" diklik maka halaman dengan nomor urut sebelumnya akan muncul, begitu juga dengan tombol "Next" diklik maka halaman dengan nomor urut setelahnya yang akan muncul.
